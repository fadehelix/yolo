# YOLO
### You Only Learn Once

Drupal 8 boilerplate theme build on node.js, gulp and sass.

## Features:
* Autoprefixer
* Sass Sourcemaps
* SC5 Style Guide Generator - just run gulp and visit http://localhost:3000.

## Requirements:
* nodejs v. 0.12
